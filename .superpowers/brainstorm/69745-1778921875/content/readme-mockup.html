⚡ AI Sales Outreach

Hyper-personalized B2B outreach — researched, written, and sent by AI agents
Python 3.11 TypeScript LangGraph n8n Claude + GPT-4 Docker FastAPI Next.js 14
Upload a CSV of leads. AI agents research each company, write a personalized email referencing their actual news and strategy, send it via Gmail, classify replies, and schedule follow-ups — automatically.
The Problem

😤 Cold outreach is broken

Generic "Hi {FirstName}, I noticed you work at {Company}" emails get a ~1% reply rate. Sales teams spend hours manually researching leads just to write emails that still feel templated.

The bottleneck isn't sending — it's the research + personalization loop that doesn't scale.

The Solution

✨ Let AI agents do the research

This platform ingests a list of leads, then spawns a LangGraph agent pipeline for each one: web research → company intelligence → hyper-personalized email → compliance check → send → reply watch → follow-up scheduling.

The product bet: personalization quality beats volume. One email that references a prospect's recent funding round outperforms 100 generic blasts.

System Architecture

🏗 Three-Plane Architecture

Each layer has a single responsibility and can be swapped independently.

Interaction Plane

Next.js + FastAPI

Dashboard, campaign management, OAuth, webhooks. Where humans and external services enter the system.

Reasoning Plane

LangGraph Agents

Every LLM call lives here. Research, personalization, reply classification, follow-up strategy.

Automation Plane

n8n Workflows

Stateless orchestration: Gmail send, Slack notify, CRM sync. No business logic — only HTTP nodes.

LangGraph Agents

🤖 The Agent Pipeline

3A · Research

Company Research Agent

Scrapes company website, news, LinkedIn. Extracts funding, strategy signals, pain points into structured JSON.

3B · Personalization

Email Personalization Agent

Generates hyper-personalized outreach using research context. Qdrant vector store for template retrieval.

3C · Reply

Reply Classifier Agent

Classifies inbound replies: interested / not-interested / out-of-office / meeting-request. Routes to CRM.

3D · Follow-up

Follow-up Strategy Agent

Selects follow-up timing and tone based on prior thread. Conditional LangGraph graph with strategy nodes.
